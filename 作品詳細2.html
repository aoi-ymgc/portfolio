制作目的

新しく開講する「総合キャリア支援プログラム」の受講生募集（申し込み獲得）を目的としたランディングページを作成したい。

ターゲット

転職やキャリアアップを目指している20~40代の男女

使用ツール

Figma, Photoshop, Visual Studio Code

Point

横幅を活かした「料金プラン」の一覧性と比較のしやすさ

広い画面幅を最大限に活用し、各プランの価格や機能差分を横並びで配置することで、一目で全体像を把握できるレイアウトを採用しました。ビジネスツールの導入検討において重要となる「比較」の工数を減らし、決裁者が論理的に判断できるよう情報を整理しています。無駄なスクロールやクリックを排除し、必要な情報を瞬時に届けられる一覧性の高さに注力しました。

「チャットUI」を活用した、読み疲れさせないリズム作り

狭い画面でもユーザーがストレスなく読み進められるよう、下部のコンテンツエリアに馴染み深いチャットアプリ風のインターフェースを採用しました。単調になりがちなテキスト情報を会話形式の吹き出しデザインに変換することで、視覚的なリズムを生み出し、文字の圧迫感を軽減しています。ユーザーは会話を追うような自然な感覚でスクロールでき、最後まで離脱せずに情報を取得できる構成にこだわりました。
