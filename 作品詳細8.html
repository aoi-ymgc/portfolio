制作目的

Blue Shoes Webサイトのデザインカンプをもとに、レスポンシブ対応のWebコーディングを行い、ユーザーにとって使いやすく魅力的なサイトを構築すること。

ターゲット

陸上クラブBLUE LINEのスクール生およびその保護者

使用ツール

Visual Studio Code

Point

デバイス幅に依存しない、崩れにくいレスポンシブ実装

デザインカンプ（PC/SP）の間にあるタブレットサイズ等の「中間サイズ」でも、レイアウトが崩れないよう流動的なコーディングを行いました。
単にブレークポイントで切り替えるだけでなく、clamp関数やrem/vw等の相対単位を適切に使い分けることで、あらゆる画面幅において余白やフォントサイズが美しく調整される「壊れない実装」を徹底しています。

運用・更新を想定したコンポーネント設計

「セクション見出し」「ボタン」「カード型レイアウト」など、繰り返し登場する要素を共通コンポーネントとして厳密に定義・実装しました。
将来的にページやコンテンツが増えた場合でも、クラスを付与するだけで統一されたデザインが適用されるよう設計しています。Sass（SCSS）の変数を活用して色や余白を一元管理し、修正依頼にも即座に対応できる保守性の高いコード記述を徹底しました。
